## Main features of this app. üßê
<br/>

- Sign in with Google.
- Create an account.
- Create notes and add content to them.
- Add photos to notes.
- Edit notes and their photos.
- Delete notes.
- Delete photos from notes.

## Instalation. üöÄ
<br/>

### 1. Clone the repository

```bash
 git clone https://github.com/Franklin361/journal-app
```
<br/>

### 2. Run this command to install the dependencies.

```bash
 npm install
```
<br/>

### 3. Create the **.env** file in the root of the project and add the following variables.

```js
    /* Firebase keys */

    VITE_APIKEY=
    VITE_AUTHDOMAIN=
    VITE_DATABASEURL=
    VITE_PROJECTID=
    VITE_STORAGEBUCKET=
    VITE_MESSAGINGSENDERID=
    VITE_APPID=

    /* Cloudinary keys */

    VITE_PRESET= 
    VITE_CLOUD_NAME=
    VITE_KEY=
    VITE_SECRET=
```
<br/>

### 4. Run this command to raise the development server.

```bash
 npm run dev
```
